# BachesYCachibaches

Proyecto TECSUP

Tareas por cumplir

Crear formularios con las siguientes caracteristicas

Campos del Formulario Usuarios
Nombres, Apellidos, Telefono, Celular, Direccion, Email, Contraseña y Foto de perfil
(En la funcion para modificar contraseña se añadira un campo contraseña antigua y contraseña nueva para la comprobacion de la misma, revisar la funcion update linea 138 del archivo userController ubicado en backend/api/controllers/userController.js)

Campos del Formulario Planes
Porcentaje, Tiempo y Precio

Campos del Formulario Comentarios
Comentario

Campos del Formulario Categorias
Nombre, Subcategorias:[{Nombre},{Nombre},...]

Campos del Formulario Anuncios
Titulo, Descripcion, Categoria, Subcategoria, Precio, Imagenes

Los Campos del Modelo Compra son todos datos que necesitan ser referenciados de otro documento o son generados automaticamente por lo que el CRUD de este modelo no se creara(para mas informacion revise el backend del proyecto)

Componentes por crear dentro del proyecto

Vista Destacar Anuncio

Vista Ver Mis Anuncios

Vista Ver mi perfil

Habilitar Botones para usuarios logeados y Deshabilitar los mismos para usuarios que aun no se han logeado

Componente para los comentarios de un Anuncio

Agregar posibles ideas que probablemente haya olvidado.
